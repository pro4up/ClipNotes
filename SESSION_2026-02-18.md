## Технические детали

### WhisperService — оптимизация (убраны --offset-t и --duration)

```csharp
// Было:
// args += $" --offset-t {offsetMs} --duration {durationMs}";
// Стало: передаём уже нарезанный аудиоклип напрямую
var args = $"-m \"{_modelPath}\" -f \"{audioClipPath}\" -t {threads}";
```

### LogService — путь к логам (исправлен для single-file publish)

```csharp
// Было: Assembly.GetExecutingAssembly().Location
// Стало:
private static readonly string LogDir = Path.Combine(
    Path.GetDirectoryName(Process.GetCurrentProcess().MainModule?.FileName) ?? ".", "Logs");
```

### ModelOption record с VRAM

```csharp
public ModelOption[] ModelOptions { get; } =
```

- `System.Windows.Forms.NotifyIcon` с контекстным меню (Открыть / Выход)
- `--tray` аргумент для запуска скрытым (используется при автозапуске с Windows)
- Реестр: `HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run\ClipNotes`
