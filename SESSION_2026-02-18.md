- Обновлён до whisper.cpp **v1.8.3** (был v1.7.4)
- Добавлен параметр `-Backend cpu|cuda`
  - `cpu` (default): OpenBLAS — оптимизированный CPU, работает везде
  - `cuda`: NVIDIA CUDA 12, требует CUDA Runtime
- `build.ps1`: добавлен параметр `-Backend`
- **Примечание:** Vulkan (AMD/Intel GPU) нет в pre-built — нужна сборка из исходников с `-DGGML_VULKAN=ON`
