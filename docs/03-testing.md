# 03 — Тестирование функционала

## Предусловия

- Приложение собрано: `app\ClipNotes.exe` существует
- OBS Studio запущен с WebSocket на `172.28.64.1:4455` (без пароля)
- В настройках ClipNotes: Host = `172.28.64.1`, Port = `4455`, OutputDirectory = `E:\Claude Workstation\Projects\ClipNotes\sessions\`
- Тестовое видео: `E:\Claude Workstation\Projects\video\2026-02-17_01-12-07\video\recording.mkv` (126.9 сек)

> **Важно:** ClipNotes — WPF-приложение, запускается только на Windows. AI-агент не может выполнить полное GUI-тестирование; только пользователь может запустить и проверить приложение.

---

## Чек-лист: OBS подключение

- [ ] Запустить `ClipNotes.exe`
- [ ] Настройки → Host = `172.28.64.1`, Port = `4455`, Password пустой
- [ ] Нажать «Проверить соединение» → статус: «Подключено к OBS»
- [ ] Лог `app\Logs\YYYY-MM-DD.log`: нет ошибок авторизации SHA256

**Если баг:** `ObsWebSocketService.cs` → метод аутентификации, challenge-response.

---

## Чек-лист: Запись с OBS

- [ ] Вкладка «Запись» → ввести имя сессии (или оставить пустым)
- [ ] Если `AskSessionName=true` — появится диалог InputDialog при старте записи
- [ ] Нажать «Начать запись» → таймер пошёл, статус «Запись...»
- [ ] Добавить маркеры через кнопки Bug / Task / Note / Summary
- [ ] Добавить маркер через горячие клавиши (Ctrl+F1–F4) из другого окна
- [ ] Нажать «Остановить запись» → приложение перешло на вкладку Экспорт
- [ ] В папке сессии (`sessions\YYYY-MM-DD.../video/`) появился видеофайл

---

## Чек-лист: Hold Mode

- [ ] Настройки → Режим удержания → включить
- [ ] Удерживать кнопку Bug (зажать ЛКМ) → появился индикатор с таймером и пульсацией
- [ ] Отпустить → маркер создан с `HoldDuration > 0`
- [ ] Удерживать горячую клавишу (Ctrl+F1) — аналогично
- [ ] В Excel после генерации: у hold-маркеров столбец длительности заполнен

---

## Чек-лист: Ручной режим (без OBS)

- [ ] Вкладка «Экспорт» → кнопка «Обзор» → выбрать тестовое видео
- [ ] Видео загружено, статус «Видео загружено: recording.mkv (00:02:06)»
- [ ] Ввести тайм-код `00:00:10`, выбрать тип Bug, нажать «+ Добавить»
- [ ] Маркер появился в списке с индексом 1, таймкод сдвинулся на +30с
- [ ] Нажать ✕ для удаления → индексы переиндексированы
- [ ] Ввести некорректный тайм-код (`abc`, `999`) → статус «Неверный формат»
- [ ] Нажать «Импорт маркеров» → выбрать JSON-файл → маркеры загружены

---

## Чек-лист: Постобработка (Generate)

- [ ] Включить «Аудио» и «Текст» для нескольких маркеров
- [ ] Нажать «Генерировать»
- [ ] Прогресс-бар заполняется, статус обновляется
- [ ] После завершения: `sessions\.../audio/` содержит аудио-клипы (без master.wav в кастомном пути)
- [ ] `sessions\.../txt/` содержит транскрипции для помеченных маркеров
- [ ] `sessions\.../table/ClipNotes.xlsx` открывается, гиперссылки работают
- [ ] Нажать «Генерировать» дважды быстро → второй вызов игнорируется (только один pipeline)
- [ ] Нажать «Отмена» во время генерации → pipeline остановлен

**Edge-cases:**
- [ ] Все маркеры с `GenerateAudio=false` и `GenerateText=false` → статус «Нет маркеров для обработки»
- [ ] Генерация без загруженного видео → статус «Сначала загрузите видео»

## Чек-лист: Кастомные пути

- [ ] Настройки → «Кастомный путь для аудио» → включить, указать папку, режим «Копировать»
- [ ] Запустить генерацию → файлы появились и в стандартной папке, и в кастомной (без master.wav)
- [ ] Режим «Переместить» → файлы только в кастомной папке, в стандартной — нет

## Чек-лист: История сессий

- [ ] Вкладка «История» → показывает прошлые записи
- [ ] Кнопка «Открыть» → открывает папку в Explorer
- [ ] «Очистить историю» → список очищается; если `DeleteFilesOnClear=true` — папки удалены

## Чек-лист: Настройки

- [ ] Поля OBS Path и OutputDirectory: можно очистить и ввести новое значение (не `IsReadOnly`)
- [ ] Горячие клавиши: кликнуть поле → нажать комбинацию → сохранилась
- [ ] Escape в поле хоткея → сброс на «Не назначено»
- [ ] Нав-кнопки в Настройках: «Горячие клавиши» → прокручивает к соответствующей карточке
- [ ] Авто-суффикс: выбрать пресет из списка → поле обновилось; «Произвольный» → поле остаётся редактируемым
- [ ] Сменить язык ru ↔ en → все строки интерфейса переключились
- [ ] Тема: Светлая / Тёмная → интерфейс и заголовок окна перекрашиваются

## Ручная проверка инструментов (PowerShell)

```powershell
# Проверить наличие всех компонентов
$app = 'E:\Claude Workstation\Projects\ClipNotes\app'
@("$app\ClipNotes.exe", "$app\tools\ffmpeg.exe",
  "$app\tools\ffprobe.exe", "$app\tools\whisper-cli.exe") | ForEach-Object {
    if (Test-Path $_) { Write-Host "OK: $_" -ForegroundColor Green }
    else { Write-Host "MISSING: $_" -ForegroundColor Red }
}
$model = Get-ChildItem "$app\models\ggml-*.bin" -ErrorAction SilentlyContinue
if ($model) { Write-Host "OK: $($model.FullName)" -ForegroundColor Green }
else { Write-Host "MISSING: models\ggml-*.bin" -ForegroundColor Red }

# Получить длительность тестового видео
& "$app\tools\ffprobe.exe" -v error -show_entries format=duration `
    -of default=noprint_wrappers=1:nokey=1 `
    'E:\Claude Workstation\Projects\video\2026-02-17_01-12-07\video\recording.mkv'

# Нарезать аудио-клип (5 сек с 10-й секунды)
& "$app\tools\ffmpeg.exe" -i 'E:\Claude Workstation\Projects\video\2026-02-17_01-12-07\video\recording.mkv' `
    -ss 10 -t 5 -vn -acodec pcm_s16le -ar 16000 -ac 1 'C:\temp\test.wav'

# Транскрибировать клип
& "$app\tools\whisper-cli.exe" -m "$app\models\ggml-base.bin" `
    --output-txt -of 'C:\temp\test_out' 'C:\temp\test.wav'
# Создаст: C:\temp\test_out.txt
```
