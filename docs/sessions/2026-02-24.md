## Изменения

### 1. Иерархия директорий установки (`app/` subdirectory)

**Было:** все файлы приложения в корне `installDir\` (`ClipNotes.exe`, DLL, tools/, models/, lang/)

**Стало:** все файлы в `installDir\app\`

```
ClipNotes\                  ← installDir (InstallLocation в реестре)
└── app\                    ← appDir (ClipNotes.exe, Uninstaller.exe, DLL, tools/, models/, lang/)
```

**Затронутые файлы:**
- `build.ps1` — добавлена переменная `$appDir`, publish нацелен на `$appDir`; ZIP-бандлы используют prefix `ClipNotes/app/` (online) и `app/` (offline)
- `ClipNotes.Setup/Services/InstallerService.cs` — `InstallAsync` создаёт `appDir`; shortcut/startup/uninstall registry указывают на `appDir/ClipNotes.exe`; `RegisterUninstaller(installDir, appDir)` — `InstallLocation` = корень, `DisplayIcon`/`UninstallString` = appDir; `GetLocalSourceDir()` проверяет `../app/app/ClipNotes.exe`; `ExtractEmbeddedBundleAsync` — case `"app"` → `appDir`

**Причина:** чистота структуры установки; все бинарники и ресурсы изолированы от корня.

---

### 2. Форматы авто-суффикса даты

**Было:** 5 пресетов — HMS (`HH:mm:ss`), HM (`HH:mm`), DM, MY, DMY + Custom

**Стало:** 3 пресета — DM (`дд.ММ`), MY (`ММ.гггг`), DMY (`дд.ММ.гггг`) + Custom

**Затронутые файлы:**
- `ClipNotes/Views/MainWindow.xaml` — удалены `ComboBoxItem` с тегами `{HH}:{mm}:{ss}` и `{HH}:{mm}`
- `ClipNotes/lang/ru/lang.json` — удалены ключи `loc_DatePreset_HMS`, `loc_DatePreset_HM`; обновлён `loc_LabelDateFormat`
- `ClipNotes/lang/en/lang.json` — аналогично

**Причина:** секунды и минуты в имени файла избыточны; для имён папок/файлов важна только дата.

---

### 3. Исправления offline ZIP (попутно с иерархией)

Ранее offline-бандл в `build.ps1` использовал обратный слэш в именах записей ZIP (`app\file.exe`). При распаковке `ExtractEmbeddedBundleAsync` разбивал имя по `/` и не находил папку `app` — файлы игнорировались.

**Исправление:**
- `build.ps1`: прямые слэши во всех именах записей offline ZIP
- `InstallerService.cs`: `entry.FullName.Replace('\\', '/')` перед разбором

## Примечание: ключ AppListBackup в реестре

Ключ `HKCU\Software\Microsoft\Windows\CurrentVersion\AppListBackup\ListOfEventDrivenBackedUpTiles_...` — создаётся Windows для резервного копирования тайлов Start Menu. Не имеет отношения к ClipNotes, удалять его деинсталлятором не нужно.
