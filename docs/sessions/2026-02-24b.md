### 3 & 4. Исправлен деинсталлятор (проблемы 3 и 4)

**Файл**: `ClipNotes.Uninstaller/MainWindow.xaml.cs`

**Причины**:
- `UseShellExecute = false` — cmd.exe запускался без elevated-токена, `rmdir` не мог удалить из Program Files
- `ping 127.0.0.1 -n 8` = 7 секунд задержки

**Исправлено**:
```csharp
var cmdExe = Path.Combine(Environment.SystemDirectory, "cmd.exe");
Process.Start(new ProcessStartInfo
{
    FileName      = cmdExe,
    Arguments     = $"/c timeout /t 4 /nobreak > nul & rmdir /s /q \"{installDir}\"",
    WindowStyle   = ProcessWindowStyle.Hidden,
    UseShellExecute = true,  // наследует elevated-токен → rmdir работает в Program Files
});
```

Задержка уменьшена с ~7с до ~4с.

### Bagfixes build.ps1

**Bug: Move-Item race condition** — при повторной сборке PowerShell `Move-Item src dest` помещает `src` ВНУТРЬ `dest` если `dest` ещё не удалилась физически с диска. В результате получалось `app/lang/lang/en/lang.json`.

Исправлено: `Copy-Item -Recurse` + `Remove-Item` вместо `Move-Item`.
