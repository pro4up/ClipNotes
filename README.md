# ClipNotes

Десктопное Windows-приложение для захвата таймкодированных заметок во время записи в OBS Studio. Помечайте баги, задачи и заметки горячими клавишами прямо во время сессии — после записи ClipNotes сгенерирует Excel-таблицу, аудио-клипы и транскрипции для каждого маркера.

## Возможности

- **OBS интеграция** — WebSocket v5; старт/стоп записи из приложения; автоподключение и автозапуск OBS
- **4 типа маркеров**: Bug / Task / Note / Summary — с точным таймкодом из OBS
- **Режим удержания** — длительность аудио-клипа определяется временем удержания горячей клавиши (или кнопки)
- **Ручной режим** — работает без OBS; загрузите готовое видео и добавьте маркеры вручную
- **Импорт маркеров** — из JSON-файла предыдущей сессии
- **Глобальные горячие клавиши** — настраиваемые комбинации для всех действий
- **Аудио-клипы** — FFmpeg вырезает клип для каждого маркера (настраиваемое окно до/после)
- **Транскрипция** — whisper.cpp с поддержкой глоссария и выбором языка
- **Excel-отчёт** — гиперссылки на аудио и текст, цвет строки по типу маркера
- **История сессий** — список прошлых записей с быстрым доступом к папке
- **Именование записей** — запрос имени при старте + авто-суффикс с датой/временем
- **Кастомные пути** — отдельные папки для video/audio/txt/table (режимы: перемещение/копирование)
- **Локализация** — ru/en; добавьте свой язык без пересборки
- **Тёмная / светлая тема** — автодетекция или ручной выбор
- **Трей и автозапуск** — минимизация в системный трей, запуск с Windows

## Установка

### Вариант 1 — Установщик (рекомендуется)

Скачайте `ClipNotes-Setup.exe` с [GitHub Releases](../../releases) и запустите.

Установщик предложит:
- Папку установки
- Бэкенд whisper: **CPU** (работает везде) или **CUDA** (быстрее, нужна NVIDIA GPU)
- Модель Whisper: base / small / medium / large-v3-turbo / large-v3

Создаёт ярлыки на рабочем столе и в меню «Пуск», регистрирует деинсталлятор в «Программах и компонентах».

### Вариант 2 — Сборка из исходников

Требования: Windows 10/11 x64, .NET 8 SDK, PowerShell 7+

```powershell
git clone <repo-url>
cd source
.\build.ps1                      # Скачивает FFmpeg, whisper-cli, модель; собирает в app/
.\build.ps1 -SkipDependencies    # Только сборка (инструменты уже есть)
.\build.ps1 -BuildSetup          # + собрать установщик → Setup/ClipNotes-Setup.exe
.\build.ps1 -BuildPortable       # + portable ZIP → Setup/ClipNotes-portable.zip
```

Результат: `app/ClipNotes.exe` — готовое к запуску приложение.

## Настройка OBS

1. Откройте OBS Studio
2. **Инструменты → Настройки сервера WebSocket** (Tools → WebSocket Server Settings)
3. Включите **Enable WebSocket Server**, порт по умолчанию: **4455**
4. При необходимости установите пароль (необязательно)
5. Нажмите **OK**

> Если пункт меню отсутствует — обновите OBS до версии 28+ (WebSocket встроен начиная с v28).

## Рабочий процесс

### С OBS

1. **Вкладка «Запись»** — нажмите «Начать запись» (или `Ctrl+F9`)
2. Во время записи нажимайте горячие клавиши или кнопки **Bug / Task / Note / Summary**
3. Нажмите «Остановить запись» — видео переместится в папку сессии
4. **Вкладка «Экспорт»** — включите/отключите аудио и текст для каждого маркера, нажмите «Генерировать»
5. Нажмите «Открыть папку» для доступа к результатам

### Режим удержания (Hold Mode)

Включите **Режим удержания** в настройках. Теперь удерживайте горячую клавишу маркера — пока клавиша зажата, приложение записывает момент «начала» события. При отпускании длительность аудио-клипа = время удержания + настройки до/после.

Работает как для горячих клавиш, так и для кнопок маркеров в интерфейсе (удерживайте ЛКМ).

### Без OBS (ручной режим)

1. **Вкладка «Экспорт»** → **«Обзор»** — выберите готовый видеофайл
2. Введите тайм-код (ЧЧ:ММ:СС) и тип, нажмите **«+ Добавить»** для каждого маркера
3. Нажмите **«Генерировать»**

## Структура сессии

```
<корневая_директория>/<YYYY-MM-DD_HH-mm-ss>/
├── video/          # Видеозапись (recording.mkv или именованный файл)
├── audio/          # master.wav + клип_001_Bug.wav / .mp3 / .aac
├── txt/            # Транскрипции: клип_001_Bug.txt
├── table/          # ClipNotes.xlsx
└── meta/           # session.json (маркеры и метаданные сессии)
```

## Горячие клавиши по умолчанию

| Действие | Комбинация |
|---|---|
| Маркер: Bug | Ctrl + F1 |
| Маркер: Task | Ctrl + F2 |
| Маркер: Note | Ctrl + F3 |
| Маркер: Summary | Ctrl + F4 |
| Начать запись | Ctrl + F9 |
| Остановить запись | Ctrl + F10 |
| Генерация | Ctrl + F11 |
| Открыть папку | Ctrl + F12 |

Все комбинации настраиваются в **Настройки → Горячие клавиши**.

## Модели Whisper

| Модель | Размер | VRAM | Точность |
|---|---|---|---|
| base | ~150 MB | ~1 GB | Базовая |
| small | ~490 MB | ~2 GB | Средняя |
| medium | ~1.5 GB | ~5 GB | Хорошая |
| large-v3-turbo | ~1.6 GB | ~6 GB | Высокая (рекомендуется) |
| large-v3 | ~3.1 GB | ~10 GB | Максимальная |

При использовании CPU-бэкенда VRAM не требуется (используется оперативная память).

## Локализация

Файлы локализации в папке `lang/` рядом с `ClipNotes.exe`:

```
lang/
├── ru/lang.json   ← русский (по умолчанию)
├── en/lang.json   ← английский
└── <код>/lang.json ← пользовательский язык
```

Чтобы добавить новый язык: скопируйте `lang/en/lang.json` в `lang/<код>/lang.json`, переведите значения (ключи не трогать), перезапустите ClipNotes — новый язык появится в настройках.

## Известные ограничения

- Только Windows 10/11 x64 (WPF)
- OBS с WebSocket v5 (версия 28+)
- CUDA-бэкенд whisper требует NVIDIA GPU; Vulkan — сборка из исходников whisper.cpp с `-DGGML_VULKAN=ON`

---

## Устранение неполадок

| Симптом | Решение |
|---|---|
| «Не удалось подключиться к OBS» | Убедитесь, что OBS запущен и WebSocket включён |
| Ошибки транскрипции | Проверьте наличие файла модели в папке `models/` |
| Нет аудио/видео | Проверьте `ffmpeg.exe` и `ffprobe.exe` в папке `tools/` |
| Горячие клавиши не работают | Конфликт с другими приложениями; смените комбинации в настройках |
| Приложение не запускается | Проверьте логи в `Logs/YYYY-MM-DD.log` рядом с `ClipNotes.exe` |
