# ClipNotes

Десктопное Windows-приложение для автоматизированного захвата таймкодированных заметок (баги/задачи/заметки) во время записи OBS. После записи генерирует Excel-таблицу, аудио-сниппеты и текстовые транскрипции.

## Возможности

- **Интеграция с OBS** через WebSocket v5 — старт/стоп записи, получение таймкодов; авто-подключение при запуске если хост/порт заполнены
- **3 типа маркеров** (Bug / Task / Note) с точным таймкодом из OBS
- **Ручное добавление маркеров** — загрузка существующего видео без OBS
- **Глобальные горячие клавиши** — настраиваемые комбинации для всех действий
- **Генерация аудио-клипов** для каждого маркера (с настраиваемым окном до/после)
- **Транскрипция** через whisper.cpp (поддержка глоссария для точности)
- **Excel-отчёт** с гиперссылками на файлы (относительные пути)
- **История сессий** с быстрым доступом к папкам
- **Тёмная / светлая тема** в стиле Apple
- **Трей** — минимизация в системный трей, автозапуск с Windows
- **Автозапуск OBS** — запускает OBS Studio при старте приложения

## Требования

- Windows 10/11 x64
- OBS Studio с включённым WebSocket сервером (версия 28+)
- (При сборке из исходников) .NET 8 SDK, PowerShell 7+

## Настройка OBS

1. Откройте OBS Studio
2. Меню **Инструменты → Настройки сервера WebSocket** (Tools → WebSocket Server Settings)
3. Включите **Enable WebSocket Server**
4. Порт по умолчанию: **4455**
5. При необходимости установите пароль (необязательно)
6. Нажмите **OK**

> Если пункт меню отсутствует, обновите OBS до версии 28+ (WebSocket встроен начиная с v28).

## Запуск

### Из готовой сборки

1. Перейдите в `app/`
2. Запустите `ClipNotes.exe`
3. На вкладке **Настройки**:
   - Укажите хост/порт OBS (по умолчанию `localhost:4455`)
   - Нажмите **Проверить соединение**
   - Выберите выходную директорию
   - Настройте горячие клавиши, параметры аудио и транскрипции

### Сборка из исходников

```powershell
cd source
.\build.ps1
```

Опции:
- `-SkipDependencies` — не скачивать FFmpeg и whisper-cli
- `-SkipModel` — не скачивать модель Whisper
- `-Model large-v3` — скачать конкретную модель

Результат: `app/` — готовое к запуску приложение.

## Рабочий процесс

### С OBS (основной режим)

1. Вкладка **Запись** — нажмите «Начать запись» (OBS начнёт записывать)
2. Во время записи нажимайте горячие клавиши или кнопки Bug/Task/Note для добавления маркеров
3. Нажмите «Остановить запись» — видео будет перемещено в папку сессии
4. Вкладка **Экспорт** — включите/отключите аудио и текст для каждого маркера, нажмите «Генерировать»
5. Откройте папку сессии для доступа ко всем файлам

### Без OBS (ручной режим)

1. Вкладка **Экспорт** → кнопка **«Обзор»** — выберите готовый видеофайл
2. Введите тайм-код и тип, нажмите **«+ Добавить»** для каждого маркера
3. Нажмите **«Генерировать»**

## Структура сессии

```
<корневая_директория>/<YYYY-MM-DD_HH-mm-ss>/
├── video/          # Видеозапись
├── audio/          # master.wav + клипы для каждого маркера
├── txt/            # Транскрипции для каждого маркера
├── table/          # ClipNotes.xlsx
└── meta/           # session.json (метаданные)
```

## Горячие клавиши по умолчанию

| Действие | Комбинация |
|---|---|
| Маркер: Bug | Ctrl + F1 |
| Маркер: Task | Ctrl + F2 |
| Маркер: Note | Ctrl + F3 |
| Начать запись | Ctrl + F9 |
| Остановить запись | Ctrl + F10 |
| Генерация | Ctrl + F11 |
| Открыть папку | Ctrl + F12 |

## Модели Whisper

| Модель | VRAM | Точность |
|---|---|---|
| base | ~1 GB | Низкая |
| small | ~2 GB | Средняя |
| medium | ~5 GB | Хорошая |
| large-v3-turbo | ~6 GB | Высокая (рекомендуется) |
| large-v3 | ~10 GB | Максимальная |

## Настройки

Хранятся в `%APPDATA%\ClipNotes\settings.json`.

| Категория | Параметр | По умолчанию |
|---|---|---|
| OBS | Host, Port, Password | localhost:4455 |
| Вывод | Корневая папка | (пусто) |
| Аудио | Секунд до/после маркера | 5 / 5 |
| Аудио | Кодек (wav/mp3/aac) | wav |
| Транскрипция | Модель Whisper | large-v3-turbo |
| Транскрипция | Язык | auto |
| Горячие клавиши | 7 действий | Ctrl+F1–F12 |
| OBS | Автозапуск OBS.exe | выкл |
| Система | Автозапуск с Windows | выкл |
| Система | Минимизировать в трей | выкл |
| Интерфейс | Тема | Светлая |

## Известные ограничения

- Приложение работает только на Windows x64 (WPF)
- Требуется OBS с WebSocket v5 (версия 28+)
- Транскрипция использует CPU или GPU в зависимости от сборки whisper.cpp
- При первом запуске build.ps1 скачивание модели (~1.6 ГБ для large-v3-turbo) может занять время

## Устранение неполадок

- **«Не удалось подключиться к OBS»** — убедитесь, что OBS запущен и WebSocket включён
- **Ошибки транскрипции** — проверьте наличие файла модели в папке `models/`
- **Нет аудио/видео** — проверьте наличие `ffmpeg.exe` и `ffprobe.exe` в папке `tools/`
- **Горячие клавиши не работают** — возможен конфликт с другими приложениями; измените комбинации в настройках
- **Логи** — в папке `Logs/YYYY-MM-DD.log` рядом с приложением
