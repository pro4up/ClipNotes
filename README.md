# ClipNotes

Десктопное Windows-приложение для автоматизированного захвата таймкодированных заметок (баги/задачи/заметки) во время записи OBS. После записи генерирует Excel-таблицу, аудио-сниппеты и текстовые транскрипции.

## Возможности

- **Интеграция с OBS** через WebSocket v5 — старт/стоп записи, получение таймкодов
- **3 типа маркеров** (Bug / Task / Note) с точным таймкодом из OBS
- **Глобальные горячие клавиши** — настраиваемые комбинации для всех действий
- **Генерация аудио-клипов** для каждого маркера (с настраиваемым окном до/после)
- **Транскрипция** через whisper.cpp (поддержка глоссария для точности)
- **Excel-отчёт** с гиперссылками на файлы (относительные пути)
- **История сессий** с быстрым доступом к папкам

## Требования

- Windows 10/11 x64
- OBS Studio с включённым WebSocket сервером
- (При сборке из исходников) .NET 8 SDK, CMake, Visual Studio Build Tools

## Настройка OBS

1. Откройте OBS Studio
2. Меню **Инструменты → Настройки сервера WebSocket** (Tools → WebSocket Server Settings)
3. Включите **Enable WebSocket Server**
4. Порт по умолчанию: **4455**
5. При необходимости установите пароль (необязательно)
6. Нажмите **OK**

> Если пункт меню отсутствует, обновите OBS до версии 28+ (WebSocket встроен начиная с v28).

## Запуск

### Из готовой сборки

1. Перейдите в `compile/ClipNotes-win-x64/`
2. Запустите `ClipNotes.exe`
3. На вкладке **Настройки**:
   - Укажите хост/порт OBS (по умолчанию `localhost:4455`)
   - Нажмите **Проверить соединение**
   - Выберите выходную директорию
   - Настройте горячие клавиши, параметры аудио и транскрипции

### Сборка из исходников

```powershell
cd source/ClipNotes
.\build.ps1
```

Опции:
- `-SkipDependencies` — не скачивать FFmpeg и whisper-cli
- `-SkipModel` — не скачивать модель Whisper
- `-Model large-v3` — скачать конкретную модель

Результат: `compile/ClipNotes-win-x64/` — готовое к запуску приложение.

## Рабочий процесс

1. **Настройки** — подключитесь к OBS, выберите директорию, настройте параметры
2. **Запись** — нажмите «Начать запись» (OBS начнёт записывать)
3. Во время записи нажимайте горячие клавиши или кнопки Bug/Task/Note для добавления маркеров
4. **Остановите запись** — видео будет перемещено в папку сессии
5. **Экспорт** — включите/отключите аудио и текст для каждого маркера, нажмите «Генерировать»
6. Откройте папку сессии для доступа ко всем файлам

## Структура сессии

```
<корневая_директория>/<YYYY-MM-DD_HH-mm-ss>/
├── video/          # Видеозапись
├── audio/          # master.wav + клипы для каждого маркера
├── txt/            # Транскрипции для каждого маркера
├── table/          # ClipNotes.xlsx
└── meta/           # session.json (метаданные)
```

## Горячие клавиши по умолчанию

| Действие | Комбинация |
|---|---|
| Маркер: Bug | Ctrl + F1 |
| Маркер: Task | Ctrl + F2 |
| Маркер: Note | Ctrl + F3 |
| Начать запись | Ctrl + F9 |
| Остановить запись | Ctrl + F10 |
| Генерация | Ctrl + F11 |
| Открыть папку | Ctrl + F12 |

## Известные ограничения

- Приложение работает только на Windows (WPF)
- Требуется OBS с WebSocket v5 (версия 28+)
- Транскрипция больших фрагментов может занимать время в зависимости от модели
- При первом запуске build.ps1 скачивание модели (~3 ГБ для large-v3-turbo) может занять значительное время
- whisper-cli требует наличия Visual Studio Build Tools для сборки из исходников

## Устранение неполадок

- **«Не удалось подключиться к OBS»** — убедитесь, что OBS запущен и WebSocket включён
- **Ошибки транскрипции** — проверьте наличие файла модели в папке `models/`
- **Нет аудио/видео** — проверьте наличие `ffmpeg.exe` и `ffprobe.exe` в папке `tools/`
- **Горячие клавиши не работают** — возможен конфликт с другими приложениями; измените комбинации в настройках
